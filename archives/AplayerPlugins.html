侧边栏音乐播放器【基于新版Next7.8】

发表于 2020-07-20 更新于 2020-07-21 分类于 Github ， blog建站相关 阅读次数： 阅读次数： Valine：
本文字数： 1.4k 阅读时长 ≈ 1 分钟

前言：

主要是更新了Next主题到最新版本7.8之后，网上找到的把播放器放在侧边栏的方法都没办法实现，因为新版的文件和目录有了变化。故作此记录。

音乐播放器Aplayer插件

  1. 插件安装

    使用 npm:

    使用 Yarn:

    • 详细见官方文档：官方中文

  2. 把~\node_modules\aplayer路径下的dist复制一份到~/hexo/themes/next/source文件夹下。

  3. 在~/hexo/themes/next/source/dist文件夹中新建music.js文件，将以下代码复制到music.js文件中：

    • const ap = new APlayer({
 container: document.getElementById('aplayer'),
 autoplay: false,
 loop: 'all',
 volume: 0.7,
 listFolded: true,
 listMaxHeight: 60,
 audio: [
     {
         name: 'name1',
         artist: 'artist1',
         url: 'url1.mp3',
         cover: 'cover1.jpg',
     },
     {
         name: 'name2',
         artist: 'artist2',
         url: 'url2.mp3',
         cover: 'cover2.jpg',
     }
 ]
});

<link rel="stylesheet" href="/dist/APlayer.min.css">
<div id="aplayer"></div>
<script type="text/javascript" src="/dist/APlayer.min.js"></script>
<script type="text/javascript" src="/dist/music.js"></script>
      • 将放置代码放在不同的~/hexo/themes/next/layout/***.swig文件内不同位置，插件就会显示在不同位置。
      • 经过调试，改版后的放置友链的相关代码已经移动到了~\themes\next\layout\_partials\sidebar路径之下的site-overview.swig里面。
        所以，找到放置友链的相关代码if theme.links，将上述放置代码添加到endif之后。播放器也就会显示在友链之后了。放在侧边栏的其他位置同理。我是把它放在if theme.links之前，endif之后，即友情链接之上，有点分割的作用吧(●ˇ∀ˇ●)
